Campus library

dévellopeur projet: Clément

Campus Library est une petite application web en Go qui permet de voir une liste de livres, consulter les détails de chaque livre et accéder à une page de contact.


Comment y accéder:

1. Cloner le lien du repo: git clone <URL_DU_PROJET>
2. Se déplacer dans le dossier: cd <NOM_DU_REPO>
3. Pour initier le programme utiliser cette commande: go mod init (car il est dans le fichier .gitignore)
4. Lancer le programme avec cette commande: go run main.go
5. Entrer cette url sur votre navigateur: http://localhost:8080
